# @vue-aria/virtualizer

Composable helpers aligned with React Aria virtualizer behavior.

## Exports

- `useVirtualizer`
- `useScrollView`
- `useVirtualizerItem`
- `layoutInfoToStyle`
- `getRTLOffsetType`
- `getScrollLeft`
- `setScrollLeft`

```ts
import { getScrollLeft, useVirtualizerItem } from "@vue-aria/virtualizer";

const left = getScrollLeft(container, "rtl");
```
