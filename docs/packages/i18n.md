# @vue-aria/i18n

Locale and direction primitives.

## `provideI18n`

Provides locale and text direction to child hooks/components.

## `useLocale`

Returns the current locale and direction (`ltr` or `rtl`).

## `useDefaultLocale`

Returns detected locale/direction when no provider is present.

## `isRTL`

Utility for determining direction from a locale string.

```ts
import { provideI18n, useLocale, isRTL } from "@vue-aria/i18n";

provideI18n({ locale: "ar-EG" });
const locale = useLocale();

isRTL("he-IL"); // true
```

### Behavior

- Inherits locale values through nested providers.
- Supports direction override independent of locale.
- Defaults to browser locale when no provider is configured.
