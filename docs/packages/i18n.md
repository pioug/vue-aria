# @vue-aria/i18n

`@vue-aria/i18n` ports locale-aware formatting helpers from upstream `@react-aria/i18n`.

## Implemented modules

- `I18nProvider`
- `useLocale`
- `useMessageFormatter`
- `useLocalizedStringFormatter`
- `useLocalizedStringDictionary`
- `useListFormatter`
- `useDateFormatter`
- `useNumberFormatter`
- `useCollator`
- `useFilter`
- `isRTL`
- `getPackageLocalizationScript`

## Upstream-aligned examples

### useLocale + useDateFormatter

```vue
<script setup lang="ts">
import { useDateFormatter, useLocale } from "@vue-aria/i18n";

const { locale, direction } = useLocale();
const dateFormatter = useDateFormatter({ dateStyle: "long" });

const label = dateFormatter.format(new Date("2026-02-13T00:00:00.000Z"));
</script>

<template>
  <div :dir="direction">{{ locale }}: {{ label }}</div>
</template>
```

### useNumberFormatter

```vue
<script setup lang="ts">
import { useNumberFormatter } from "@vue-aria/i18n";

const currency = useNumberFormatter({ style: "currency", currency: "USD" });
</script>

<template>
  <span>{{ currency.format(1234.56) }}</span>
</template>
```

### useMessageFormatter

```vue
<script setup lang="ts">
import { useMessageFormatter } from "@vue-aria/i18n";

const formatMessage = useMessageFormatter({
  "en-US": {
    welcome: "Welcome, {name}",
  },
});
</script>

<template>
  <p>{{ formatMessage("welcome", { name: "Ada" }) }}</p>
</template>
```
